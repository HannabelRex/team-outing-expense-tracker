# Login Font Contrast Hotfix

This hotfix improves the signed-out login screen typography contrast and visual hierarchy.

## Changes

- Adds a blue/emerald gradient to the main form heading.
- Applies a distinct brand-blue color to the product kicker.
- Applies stronger blue label color to auth form fields such as Email and Password.
- Gives Forgot password and Back to sign in a clearer link style.
- Gives the New user/Create account switch button a blue/emerald tinted treatment.

## Impact

This is a frontend-only UI update. It does not change authentication logic, Supabase configuration, backend APIs, database data, or environment variables.
