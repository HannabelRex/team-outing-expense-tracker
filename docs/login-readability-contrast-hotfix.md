# Login Readability Contrast Hotfix

This hotfix improves the login screen contrast after the Workspace Access redesign.

## Changes

- Changed the login card to a brighter frosted surface so dark text is readable.
- Improved the Sign in/Create account heading gradient for better contrast.
- Made Email, Password, and form labels dark and bold.
- Made Forgot password and account-switch links visibly blue/green with stronger contrast.
- Improved input text and placeholder contrast.
- Kept the glassmorphism background and mobile-first layout.

No backend, database, or environment variable changes are required.
