# Expense List Description and Compact Actions Hotfix

This hotfix improves the Expenses tab list view.

## Changes

- Expense notes are now visible directly in the Expense list as a Description block under the expense title and metadata.
- Action buttons were changed from full text buttons to compact icon buttons for Edit, Approve, Reject, and Delete.
- Action buttons include accessible labels and hover titles so the compact layout remains understandable.
- Disabled icon buttons now visibly fade when expenses are locked by a submitted/approved/received claim.

## Backend impact

No backend changes are required. This is a frontend-only usability improvement.
